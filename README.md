# natural_units

Wrapper for various units:

| Unit      | Description |
| ----------- | ----------- |
| base      | general super class       |
| si        | m, kg, s, A, K, mol, cd - base        |
| natural   | $c=\h_bar=1$, checks mass dimension compatibility        |
| plain_natural       | $c=\h_bar=1$, like natural, but without mass dimension checks        |
